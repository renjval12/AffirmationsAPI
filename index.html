What is Affirmations API

Affirmations REST API allows developers to access and integrate the functionality of Affirmations with other applications. The API retrieves random affirmations to motivate all those who read them.

Why Affirmations

Affirmations is for a developer who wants an API that provides random affirmations to motivate users on their applications.

URL

        GET: https://affirmationsapi.herokuapp.com

Parameters

affirmation :

Generates a random affirmation for the user to utilize.
